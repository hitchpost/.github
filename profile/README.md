# Hitchpost

**Practical tools — and senior engineering — for production systems.**

Hitchpost is a principal-led consulting practice focused on the parts of software
people actually feel: how fast a page loads, whether a system stays up under load,
and how much it costs to run. The tools we open-source here come straight out of
that work — small, sharp utilities for checking, measuring, and understanding
systems in production.

## How we think about this

- **Measure first.** Real numbers, from the lab and the field, drive the work.
- **Small, sharp tools.** Do one thing well; compose rather than sprawl.
- **Built to hand off.** Documented and maintainable, so others can own it.
